📋 QUICK START (5 MINUTES):
─────────────────────────────────────────────────────────────────────

1. Install dependencies:
   npm install

2. Setup API keys:
   npm run setup

   - Gemini API key (FREE): https://makersuite.google.com/app/apikey
   - Freepik API key: From your Freepik account

3. Generate images:
   npm run generate-images
