STEP 1: Get FREE API Keys (3 minutes)
───────────────────────────────────────────────────────────────

A) Gemini Key:
   → Visit: https://makersuite.google.com/app/apikey
   → Click "Create API Key"
   → Copy it

B) Replicate Token:
   → Visit: https://replicate.com
   → Sign up (FREE)
   → Go to: Account → API Tokens
   → Copy token (starts with r8_)


STEP 2: Paste Keys in config.json (1 minute)
───────────────────────────────────────────────────────────────

Open file: config.json

Find this line:
  "apiKey": "PASTE_YOUR_GEMINI_KEY_HERE",

Replace with:
  "apiKey": "your-actual-gemini-key",

Find this line:
  "replicateKey": "PASTE_YOUR_REPLICATE_TOKEN_HERE"

Replace with:
  "replicateKey": "r8_your-actual-token"

Save file.


STEP 3: RUN IT! (15 minutes automated)
