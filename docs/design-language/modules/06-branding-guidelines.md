## ‚úçÔ∏è Writing Style

**Do:**

- Use active voice
- Keep sentences short (15-20 words)
- Use bullet points
- Include specific metrics
- Write for 8th-grade reading level

**Don't:**

- Use jargon without explanation
- Write long paragraphs
- Use passive voice
- Make vague claims
- Use ALL CAPS (except acronyms)
