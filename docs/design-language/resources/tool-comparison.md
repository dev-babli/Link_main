## üí° Tool Selection Guide

### Questions to Ask

**Budget?**

- Free ‚Üí Figma, Google Fonts, Lucide
- Paid ‚Üí Adobe CC, premium tools

**Team Size?**

- Solo ‚Üí Simpler tools
- Team ‚Üí Collaboration-focused

**Project Type?**

- Web ‚Üí Figma, web-based tools
- Mobile ‚Üí Consider platform-specific

**Existing Stack?**

- Adobe ecosystem ‚Üí Adobe XD
- Otherwise ‚Üí Figma

**Skill Level?**

- Beginner ‚Üí Figma, Canva
- Advanced ‚Üí Any tool
