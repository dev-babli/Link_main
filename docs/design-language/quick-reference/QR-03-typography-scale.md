## üìê Usage Guide

**Headlines:**

- Font: Orbitron Bold (700-900)
- Size: 48-60px
- Line-height: 1.1-1.2

**Subheadings:**

- Font: Inter Semibold (600)
- Size: 24-36px
- Line-height: 1.3

**Body Text:**

- Font: Inter Regular (400)
- Size: 16-18px
- Line-height: 1.5-1.6

**Small Text:**

- Font: Inter Regular (400)
- Size: 12-14px
- Line-height: 1.4-1.5
